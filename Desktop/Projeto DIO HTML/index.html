HTML (HyperText Markup Language)

(Voltar)

O HTML é a linguagem de marcação usada para estruturar o conteúdo de uma página web.

Ela define títulos, parágrafos, imagens, botões e formulários.

Usa-se tags como: <h1>, <p>, <img>, <a>, <div>, etc.

Curiosidade histórica

O HTML foi criado por Tim Berners-Lee, um cientista da computação britânico.

Ele trabalhava no CERN (Organização Europeia para Pesquisa Nuclear) e queria facilitar o compartilhamento de informações entre cientistas.

Em 1991, Berners-Lee publicou a primeira versão do HTML, junto com o primeiro site da história.

CSS (Cascading Style Sheets)

(Voltar) O CSS é responsável pela estilização das páginas, definindo cores, fontes, espaçamentos e layouts.

Serve para deixar o site bonito visualmente. Utilizando alguns seletores como: "body", ".classe", "id".

Curiosidade histórica

No início da web, os estilos eram aplicados diretamente no HTML, o que tornava os códigos longos e difíceis de modificar.

A necessidade de um método mais eficiente levou à criação do CSS.

JavaScript (JS)

(Voltar)

O JavaScript é a linguagem de programação usada para tornar o site interativo e dinâmico.

Fazendo toda a funcionalidade juntamente com o visual. Manipula elementos do HTML, adicionando eventos (cliques, animações e validações de formulários) e conectando o site a APIS.

Curiosidade histórica

Em 1995, foi criado por Brendan Eich, da empresa Netscape, em apenas 10 dias!

O objetivo era permitir interatividade nas páginas web.

Inicialmente, se chamava Mocha, depois LiveScript, até ser renomeado para JavaScript (para surfar na popularidade do Java na época).
